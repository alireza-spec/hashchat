# HashChat

پیام‌رسان پیشرفته مشابه تلگرام

ساخته شده توسط علیرضا احمدی
[@HashtagAlireza](https://instagram.com/HashtagAlireza)

## امکانات
- چت خصوصی و گروهی
- تماس صوتی و تصویری
- ارسال فایل و تصویر
- پیام‌های پین شده
- پروفایل کاربری با تم دلخواه
- جستجوی کاربران

## اجرای محلی
```bash
npm install
cd server && npm install
cd ../client && npm install
npm run dev
```
سپس http://localhost:3000 را باز کنید.

## دیپلوی
برای دیپلوی آنلاین، از سرویس‌هایی مثل Render یا Railway استفاده کنید.
